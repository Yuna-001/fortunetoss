## 📆 프로젝트 개요

- **프로젝트명:** 던지미 (Fortunetoss)
- **개발 기간:** 2025.01 (1개월)
- **주제:** 새해 덕담과 지난 한 해 동안의 자신에 대한 퀴즈를 담은 복주머니를 만들어 친구와 나누는 설날 이벤트성 웹 서비스
- **배포:** ~~[Fortunetoss 홈페이지](https://fortunetoss.vercel.app)~~ (현재 배포 중단)

<br>

## 🎯 주요 기능

**🎁 복 나누미 (퀴즈 출제자)**

- 로그인 후, 공유할 퀴즈와 덕담을 담은 복주머니 생성
- 퀴즈만 또는 퀴즈+덕담 중 하나의 옵션 선택
- 완성된 복주머니를 카카오톡 또는 링크 복사로 공유
- 복주머니별 응답자의 답변 통계 확인

**🍀 복 받으미 (퀴즈 응답자)**

- 공유받은 복주머니의 퀴즈 풀기 및 덕담 확인
- 자신의 결과를 카카오톡 또는 링크 복사로 공유
